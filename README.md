# Centrum Badań Jutra - Landing Page dla badań klinicznych

Nowoczesny, responsywny landing page dla centrum badań klinicznych. Strona prezentuje ofertę dla pacjentów oraz sponsorów/CRO, promuje aktualne programy badawcze i prowadzi użytkownika do szybkiej kwalifikacji online.

## 🔗 Live Demo

https://nubet.github.io/klinika-centrum-badan-landing-page/

## 🎯 Cel Biznesowy

- Zaprezentowanie centrum badań klinicznych jako nowoczesnego i wiarygodnego partnera dla pacjentów oraz sponsorów.
- Zachęcenie pacjentów do udziału w aktualnie prowadzonych badaniach klinicznych.
- Ułatwienie szybkiej kwalifikacji online poprzez prosty, wieloetapowy formularz.
- Zbudowanie zaufania przez jasne przedstawienie korzyści: bezpłatnej diagnostyki, dostępu do innowacyjnych terapii oraz opieki medycznej.
- Przedstawienie zaplecza technicznego i organizacyjnego dla sponsorów oraz firm CRO.
- Zapewnienie czytelnego, responsywnego interfejsu działającego poprawnie na desktopie i urządzeniach mobilnych.

## 🛠️ Stack Technologiczny

- **Framework:** React + Vite
- **Język:** TypeScript
- **Styling:** Tailwind CSS

## ⭐ Kluczowe cechy projektu

- **Responsywny layout:** Strona została przygotowana z myślą o poprawnym wyświetlaniu na różnych szerokościach ekranu.
- **Sekcja hero nastawiona na konwersję:** Wyraźny nagłówek, opis wartości oraz dwa przyciski CTA kierujące osobno do pacjentów i sponsorów.
- **Prezentacja korzyści dla pacjenta:** Karty informacyjne komunikują najważniejsze argumenty za udziałem w badaniu klinicznym.
- **Aktualne programy badawcze:** Landing page zawiera sekcję z kartami rekrutacji, podzielonymi według obszarów terapeutycznych.
- **Sekcja dla sponsorów i CRO:** Osobny blok prezentuje zaplecze centrum, szybkość rekrutacji oraz jakość danych.
- **Wieloetapowy formularz kwalifikacyjny:** Formularz prowadzi użytkownika przez krótki proces zgłoszenia i zawiera podstawową walidację danych po stronie frontendu.
- **Dodatkowe strony informacyjne:** Projekt zawiera osobne podstrony dla polityki prywatności oraz RODO.

## 📄 Struktura strony

Landing page składa się z kilku głównych sekcji:

- Nawigacja główna
- Hero z głównym komunikatem i CTA
- Korzyści udziału w badaniach klinicznych
- Aktualne rekrutacje
- Sekcja dla sponsorów i CRO
- Formularz szybkiej kwalifikacji online
- Stopka z danymi oraz linkami informacyjnymi
